How it works

Chunked Streaming

Files are split into small chunks on the client-side for reliable transmission.

Redis Caching

Each chunk is cached in Redis. If the upload breaks, the server remembers what it received.

Auto-Resume

When the connection returns, we only upload the missing chunks, saving time and bandwidth.
